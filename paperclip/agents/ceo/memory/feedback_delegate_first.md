Triage and delegate all `todo` tasks to the right agent as the FIRST action in every heartbeat, before doing any CEO-specific work (strategy, planning, admin).

**Why:** Board flagged that tasks were stalling because CEO heartbeats are ~12 min apart and I was doing analysis/deep work before delegating. Engineering and research tasks sat in my queue for 1-2 heartbeat cycles unnecessarily.

**How to apply:** After identity check and assignment fetch, immediately scan all `todo` tasks assigned to me. If a task is engineering → Founding Engineer. If research → Researcher. Delegate with a brief routing comment. Only then proceed to CEO-owned work. Never hold delegatable tasks across heartbeats.
